As a administrator, you have the right to use this application.

This application is for administrator to setting question for the location-based quiz app. Your submit question will be uploaded into database and will pop-up in the quiz app. Since this app is still developing. The function is not perfect. Functions such as question modification or drop cannot be achieved. Therefore, please set reasonable question. This guide will demonstrate the use of this application by bullet point:

1. Open the website http://developer.cege.ucl.ac.uk:31276 on Chrome, Internet Explorer or FireFox browser

2. View the exisiting question shown as blue marker by clicking the 'View Questions on Map' button, question id and question can be viewed by clicking blue marker

3. If the markers are two intensive, you can click the 'Hide Questions on Map' button to hidden the question marker

4. Fill each field: circle button is to set your single correct answer; Latitude and Longitude can be obtained by clicking your interesting location on map, in which the coordinate of that point will be displayed

5. If you want to change all the content you typed in before submission, just click 'All Clear' button. It is not the button for drop the submitted question from database

6. Click 'Upload Question' to upload your question to database

7. Alert will pop-up to remind you fill all the fields, check the circle or enter resonable number for latitude and longitude

8. Text below all the buttons will remind you the result of uploading: rows inserted or error

Thanks for using this App. Any comments and suggestions are always welcome!
